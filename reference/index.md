# Package index

## Main functions

- [`use_metadata_template()`](https://delma.ala.org.au/R/reference/use_metadata_template.md)
  : Write an example metadata statement to disk
- [`read_md()`](https://delma.ala.org.au/R/reference/read_md.md) : Read
  markdown-formatted metadata
- [`write_md()`](https://delma.ala.org.au/R/reference/write_md.md) :
  Write a markdown-formatted metadata document
- [`read_eml()`](https://delma.ala.org.au/R/reference/read_eml.md) :
  Read an EML-formatted metadata document
- [`write_eml()`](https://delma.ala.org.au/R/reference/write_eml.md) :
  Write an EML-formatted metadata document
- [`check_metadata()`](https://delma.ala.org.au/R/reference/check_metadata.md)
  : Check validity of a metadata statement

## Format manipulation

- [`as_lp_tibble()`](https://delma.ala.org.au/R/reference/as_lp_tibble.md)
  :

  Convert metadata to a lightparser `tibble`

- [`as_eml_tibble()`](https://delma.ala.org.au/R/reference/as_eml_tibble.md)
  :

  Convert metadata to a `tibble`

- [`as_eml_list()`](https://delma.ala.org.au/R/reference/as_eml_list.md)
  :

  Convert metadata to a `list`

- [`as_eml_xml()`](https://delma.ala.org.au/R/reference/as_eml_xml.md) :

  Convert metadata to an `xml_document`
